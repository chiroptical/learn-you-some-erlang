Notes

Use `rebar3 shell` to get access to `src` files, e.g. `c(event).` will load up
`src/event.erl` for you. 

Shell Notes
---

```erl
c(module). % compiles and imports a module
rr(module, record). % imports record from module
f(). % clear all variables
q(). % close the shell
```
